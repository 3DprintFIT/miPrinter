miPrinter
=========

Foldable micro RepRap printer. Not ready for production.

Rebuild
-------

To rebuild the parts, run sources/build.sh (You'll need OpenSCAD).

Print
-----

* Twice:
 * cornerL.stl
 * cornerR.stl
 * coupling.stl
 * motortop.stl
 * topholder.stl
* Once:
 * pulley32.stl
 * pulley8.stl
 * side.stl
 * side-center.stl
 * side-center.stl
 * x-carriage.stl
 * x-end-idler.stl
 * x-end-motor.stl
